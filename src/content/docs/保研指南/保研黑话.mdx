---
title: 保研黑话
---

| **术语**          | **含义**                                                                                                                                           |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| bg                | background，即自己的排名、竞赛、科研、本科学校水平等情况                                                                                           |
| rk/rank           | 指自己的排名                                                                                                                                       |
| com               | committee，指学校的招生办                                                                                                                          |
| bar               | 学校筛选学生进入面试的最低要求。因报名人数众多，不可能让每个学生都进去面试，所以需要 bar                                                            |
| wl/wait list      | 就是候补，如果前面的人选择其他学校，那么候补就有机会上位                                                                                           |
| oq/over qualified | 当某同学的实力远超学校的 bar 时，学校仍有可能拒绝该学生的申请。这种现象叫 oq，因为该生来这所学校的可能性太低，学校没必要浪费资源面试这位同学      |
| 0812              | 计算机科学与技术（学硕）                                                                                                                           |
| 0835              | 软件工程（学硕）                                                                                                                                   |
| 0839              | 网络空间安全（学硕）                                                                                                                               |
| 085400            | 电子信息类（专硕）                                                                                                                                 |
| 085404            | 计算机技术（0812 对应的专硕）                                                                                                                      |
| 085405            | 软件工程（0835 对应的专硕）                                                                                                                        |
| 085412            | 网络与信息安全（0839 对应的专硕）                                                                                                                  |
| 陶瓷or套磁        | 指向老师发邮件等方式取得私人联系，打好关系                                                                                                           |
| 羊导/颠导/强导/弱导 | 指导老师的级别，强导老师的指导力度更强（往往更加push），弱导老师的指导力度更弱（未必不push），颠导（任务强度令人不适，对待学生态度不友善），羊导（任务轻松，对待学生很热情，但往往指导很少）|
